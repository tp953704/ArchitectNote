# Rust 開發環境設置與基礎特性筆記

## Rust 關鍵特性

### 🛡️ 記憶體安全
- **獨特的所有權系統**：無需垃圾回收器
- **三大核心概念**：
  - **所有權**：每個數據都有唯一所有者
  - **借用**：創建數據引用但不擁有數據
  - **生命週期**：確保引用始終有效

### 🔄 併發處理
- 所有權模型確保多線程安全訪問數據
- 編譯時捕捉潛在問題
- 更容易編寫安全高效的併發代碼

### ⚡ 高效性能
- 設計速度與 C/C++ 相當
- 提供低級別記憶體控制
- 兼具高級語言安全性和低級語言性能

### 🛠️ 開發工具鏈
- **Cargo**：構建系統和包管理器
- **Rustfmt**：代碼格式化工具
- **Rust Analyzer**：IDE 工具，提供代碼補全和錯誤提示

## 開發環境設置

### 安裝 Rust
```bash
# 使用 Rustup 安裝（推薦）
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 驗證安裝
rustc --version
cargo --version
```
### 常用命令 
```
# 更新工具鏈
rustup update

# 打開本地文檔
rustup doc

# 檢查代碼錯誤
cargo check

# 構建項目
cargo build

# 構建並運行
cargo run
```
### IDE 設置
1. 安裝 VS Code
2. 安裝 Rust Analyzer 擴展
3. 配置開發環境

### 創建第一個 Rust 項目
1. 使用 Cargo 創建項目
```
cargo new hello-cargo
cd hello-cargo
```
2. 項目結構
```
hello-cargo/
├── Cargo.toml    # 項目配置和依賴
└── src/
    └── main.rs   # 源代碼
```
Cargo.toml 示例
```
[package]
name = "hello-cargo"
version = "0.1.0"
edition = "2021"

[dependencies]
# 依賴項在此添加
```
3. 編譯和運行
```
# 編譯項目
cargo build
# 運行可執行文件
./target/debug/hello-cargo
# 或直接編譯並運行
cargo run
```
